## 前言

#### 测试之“道”

> 1. 共享目标：快速健康将商业idea交付用户，带来价值
> 2. 质量不是QA“测”出来的, 全流程，全角色的努力，才有质量
>
> 3. 越早测试，越易发现，修复成本越小

*（一个很敬佩的领导总结的，讲得很有道理）*
